
# Teste Node com Docker e MongoDB

Projeto backend simples desenvolvido com **Node.js**, utilizando **Express**, com execução em containers via **Docker Compose** e integração com **MongoDB**.

## 🚀 Objetivo
Este projeto foi criado para praticar a criação de uma aplicação backend com containerização, organização de ambiente de desenvolvimento e execução com Docker.

## 🛠️ Tecnologias utilizadas
- Node.js
- Express
- Docker
- Docker Compose
- MongoDB
- JavaScript
